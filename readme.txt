Простой в повторении темброблок, c управлением на Arduino.
Реализован графический интерфейс на основе дисплея OLED SSD1306 0.96'
Управление и ввод при помощи инкрементального энкодера.

- Установить библиотеки из папки libraries
- Загрузить скетч
- Собрать темброблок и схему управления 

Проект на EasyEda: https://easyeda.com/Nich1con/TDA8425

Список компонентов:
>Конденсаторы
(Неполярные, керамика/пленка, 50В)
- 1 мкф 7 шт.
- 33 нф 2 шт.
- 15 нф 2шт.
- 5600 пф 2шт.
